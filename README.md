# OPEngine
A custom rendering engine written in c/c++ with openGL, mainly for implementing different lighting algorithms and rendering techniques 

# Features
- Deferred and Forward Rendering
- Normal Mapping
- Cascaded ShadowMaps
- Tonemapping
- FXAA
